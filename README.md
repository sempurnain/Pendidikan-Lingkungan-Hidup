# 🌱 Website Peta Konsep Pendidikan Lingkungan Hidup (PLH)

Website ini berisi **peta konsep Pendidikan Lingkungan Hidup (PLH)** yang dirancang untuk jenjang Sekolah Dasar (SD) / Madrasah Ibtidaiyah (MI).  
Konten disusun secara sistematis, ringkas, dan mudah dipahami untuk mendukung pembelajaran tematik serta penguatan karakter peduli lingkungan sejak dini.

---

## 📌 Tentang Website

Website ini memuat peta konsep yang terdiri dari:

1. **Pengertian Pendidikan Lingkungan Hidup (PLH)**  
   Mengacu pada sumber resmi Kementerian Lingkungan Hidup serta berbagai jurnal ilmiah yang membahas pendidikan lingkungan.

2. **Tujuan Pendidikan Lingkungan Hidup**  
   Membentuk kesadaran, sikap, dan perilaku peduli terhadap kelestarian lingkungan.

3. **Prinsip-Prinsip Pendidikan Lingkungan Hidup**  
   - Berkelanjutan  
   - Partisipatif  
   - Kontekstual  
   - Integratif dalam pembelajaran  

4. **Materi PLH untuk SD/MI**  
   - Kebersihan lingkungan  
   - Pengelolaan sampah  
   - Pelestarian sumber daya alam  
   - Hemat energi dan air  
   - Penghijauan  

5. **Metode Pembelajaran PLH**  
   - Diskusi kelompok  
   - Observasi lingkungan sekitar  
   - Project based learning  
   - Praktik langsung  

6. **Kegiatan Implementasi di Sekolah**  
   - Kerja bakti  
   - Bank sampah  
   - Penanaman pohon  
   - Kampanye hemat energi  

7. **Kesimpulan**  
   Pendidikan Lingkungan Hidup merupakan fondasi penting dalam membentuk karakter siswa yang peduli, bertanggung jawab, dan sadar akan keberlanjutan lingkungan.

---

## 🎯 Tujuan Pembuatan Website

Website ini dibuat untuk:
- Mendukung pembelajaran PLH di SD/MI
- Menyediakan media pembelajaran visual berbentuk peta konsep
- Membantu guru dan mahasiswa PGMI memahami struktur materi secara sistematis
